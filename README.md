# luchok

> 64x32 is enough for everybody.

Luchok is a Fantasy Console inspired by Chip-8.

# Features

Screen: 64x32
Scripting: Lua
Sound: 440Hz square wave
Controls: 16 buttons, labeled from 0 to F

# Building
Building luchok requires:
* cmake (>=3.0)
* libsdl2-dev
* liblua5.4-dev
